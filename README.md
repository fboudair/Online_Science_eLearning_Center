# 📘 Science eLearning Center – Capstone Project
Welcome to the Science eLearning Center – a web-based educational platform designed 
to help students learn science through interactive lessons, quizzes, and multimedia content. 
This project is part of my final web development capstone at Trios College.

## ✨ Key Features
- 📚 Science courses: Physics, Chemistry, Environmental Science
- 🧪 Interactive quizzes with scoring
- 🔐 User registration and login
- 🎯 User dashboard with course progress
- 💬 Clean, responsive design

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 8+
- **Database:** MySQL
- **Environment:** macOS, XAMPP/MAMP
- **Version Control:** Git + GitHub

## 🧪 Usage
Register a new user account

Browse lessons and complete quizzes
